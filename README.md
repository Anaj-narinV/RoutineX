# RoutineX

**RoutineX** is an customized personal tracker built to manage my daily habits, studies, health, routines, productivity, and personal growth in one place.

Every feature, section, workflow, and design choice in this project was created based on my own lifestyle, goals, and tracking requirements. This is not a generic productivity application — it is a system tailored specifically for how I plan, track, and improve my day-to-day life.

## Features

* 📅 Daily Tracking — Log routines, meals, hydration, mood, habits, and daily activities
* 🔥 Streak System — Track consistency with current and best streak records
* 📊 Analytics Dashboard — Weekly and monthly insights with charts and progress summaries
* 📝 Auto-Save Drafts — Automatically saves entries while typing
* 📚 Study Tracker — Track subjects, topics, study sessions, and study hours
* 🗒️ Planner — Manage assignments, tasks, priorities, and deadlines
* ⬇️ Data Export — Export data as JSON or CSV backups
* 📱 PWA Support — Installable on mobile and desktop with offline support
* 🎨 Premium UI — Modern dark interface with glassmorphism effects and smooth animations

## About This Project

RoutineX was built exclusively for my personal use.

The database structure, habit categories, study modules, planner workflow, analytics system, streak logic, forms, and overall user experience have been customized around my own routines and goals.

This project serves as my personal dashboard for tracking progress, maintaining consistency, and documenting my daily journey.

## Tech Stack

* Backend: Flask (Python)
* Database: SQLite
* Frontend: HTML, CSS, JavaScript
* Charts: Chart.js
* PWA: Service Worker + Web App Manifest
